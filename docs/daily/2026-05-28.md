## Key Technical Decisions

**`color-mix(in oklch, ...)` for node backgrounds**  
Rather than maintaining separate light/dark background values per node type, a single CSS expression `color-mix(in oklch, <header-color> 14%, var(--nca-color-surface))` was used. This automatically blends each node's accent color with the current theme's surface, producing natural tints across all 8 themes without any JavaScript.

**Drawer over floating panel for node inspector**  
A full-height slide-in drawer (right edge, no border-radius) was chosen over a floating rounded card because it makes the canvas feel wider when collapsed and avoids z-index conflicts. The drawer is always in the DOM; CSS `transform` handles show/hide so the transition fires in both directions.

**Two `-m` flags for Co-Authored-By**  
Discovered that `git commit -m "title\nCo-Authored-By: ..."` does not produce a proper git trailer — the newline is not interpreted. The correct pattern is `git commit -m "title" -m "Co-Authored-By: ..."`, which creates a separate paragraph that git recognizes as a trailer. Documented in three places.
